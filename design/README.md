Design kit for this project (Figma import)

Contents
- styles/tokens.json — color and typography tokens mirroring the app
- components/*.svg — basic components to reuse in Figma
- screens/*.svg — wireframe screens (Desktop + Mobile)

How to import into Figma (free)
1) Open Figma → New design file
2) File → Place image… → select the SVGs in screens/ and components/
3) For tokens, open styles/tokens.json and create Color/Text styles in Figma to match

Notes
- SVGs are vector and can be scaled. Replace placeholder text with real content.
- Use Auto Layout in Figma to convert these into interactive components.
- Colors map to Tailwind-like scales used in the codebase.
